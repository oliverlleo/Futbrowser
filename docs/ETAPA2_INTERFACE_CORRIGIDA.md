# Interface da Etapa 2 corrigida

A tela de propostas e primeiro contrato foi refeita para seguir o mockup aprovado:

- fundo noturno com estádio visível;
- cabeçalho e indicador das duas etapas;
- três colunas: propostas, dossiê e contrato;
- cards compactos de clubes com escudos locais;
- dossiê com visão geral, elenco, academia e treinador/tática;
- painel de negociação com histórico, compatibilidade e ações;
- responsividade para desktop, tablet e celular;
- integração preservada com as RPCs existentes.

Arquivos principais alterados:

- `dashboard.html`
- `src/pages/dashboard/dashboard.css`
- `src/pages/dashboard/dashboard.js`
- `src/pages/dashboard/offers-ui.js`
